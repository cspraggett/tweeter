# Tweeter Project

Tweeter is a simple, single-page Twitter clone. It was built with HTML, CSS, AJAX, and JavaScript.


## Getting Started

- To run tweeter first run: npm i
- Then run: npm start

## Dependencies

- Express
- Node 5.10.x or above
- Body-parser
- Chance
